Интерактивный баннер. По нажатию на кнопку фигурка студента делает переход по обозначенным точкам. 
Для вычисления координат перемещений я написал отдельную программу (прилагается), ее уникальность в том - что можно вычислить координаты баннера по клику мыши на соответствующие места баннера, и они будут const относительно самого баннера, при любом разрешении экрана и скроле в любую сторону (т.к. сам баннер является фиксированным по размерам).
Рейтинговая таблица строится в таблицу из формата JSON, затем автоматически группируется по записям. Можно складывать уникальные фото в папку photos, так чтобы уникальное имя из объекта совпадало с изображением (например имя == название фото файла), и таким образом соответствующие аватарки будут сразу подгружаться.
Таким образом, на примере этого простого проекта были реализованы несколько полезных программ, принцип которых можно использовать в других проектах.
